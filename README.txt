・これはなに？
  子供用のお遊び？ソフトです。

  keydemo
　　キーボードをいじるのが大好きな子供のためのソフトです。
　　がちゃがちゃ適当に叩くと何も表示されなくなり、
　　一文字ずつタイプしていくと入力した文字が表示されます。
　　だけど、手加減を知らない子供は強く叩きすぎるため、
　　ほとんど使っていません。

  mouse
　　flashみたいな。右ボタンを押すと、押している間、任意の何かが表示され、
　　離すと消えます。
　　最大10個、表示した内容を覚えており、左ボタンでさかのぼります。
　　文字だけでなくイメージも対応予定。
   
　  古いマウスを用意し、LEDを外して子供のおもちゃにしています。
    子供と遊ぶときはそのマウスを持ってこさせ、PCに接続し、
  　遊ばせます。

　　LEDを見てしまうので目に悪そう。また、意図しないところにフォーカスが
　　当たってしまうので、LEDを外しました。

　　子供に右クリックさせ、表示された文字（もしくは図形）を読み上げさせます。
　　ときには本当？と聞いて、左クリックで確認します。
